Fungi Edibility Convolutional Neural Network Application

My goal for this project was to be able to build an artifical intelligence model using a deep learning technique called Convolutional Neural Network. A Convolutional Neural Network Application is essientially a photo classifyer system. This type of deep learning technique is able to classify an image of a mushroom, that is uploaded to the system. This application specifically focuses on different types of Fungi, that are commonly found on the west coast of North America. The application I have built will then determine if the species of Fungi is apart of an either poisonous or edible genus of fungi. This application is for individuals who are not expert mycologists an strictly for educational purposes.

Analysing Customer Opinoins from Tweets

Our team's goal is to extract useful info from tweets on Apple products and explore the emotion distribution within each target product, with that emotion being either positive or negative.
By locking in on these specifications we are able to identify any pain points or repetitive results in a negatively directed manner, towards any target products and provide insight for any improvements or enhancements that Apple can move forward based on the public view.

Utilize the dataset to build a brand sentiment analysis model. Train a machine learning or deep learning model using the labeled tweets as input and their corresponding sentiment (positive or negative) as the output.
